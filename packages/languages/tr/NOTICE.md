This package includes data derived from OpenSubtitles v2024 via OPUS in
commonWords.json (https://opus.nlpl.eu/).

Source dataset: OpenSubtitles 2024
Provider: Helsinki-NLP / OPUS
Original corpus site: https://www.opensubtitles.org/

Original data is attributed to subtitle contributors and rights holders.
This package contains a processed derivative:
- tokenized text
- frequency aggregation
- normalization and filtering

Please cite the OPUS/OpenSubtitles corpus when using this data:
P. Lison and J. Tiedemann, 2016, OpenSubtitles2016: Extracting Large
Parallel Corpora from Movie and TV Subtitles. Proceedings of LREC 2016.

Any redistribution of this package must retain this notice.
