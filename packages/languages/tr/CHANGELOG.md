# 4.0.0-beta.3

Initial Turkish language package scaffold.

## Data sources

- Common words: https://object.pouta.csc.fi/OPUS-OpenSubtitles/v2024/freq/tr.freq.gz
- First names: https://raw.githubusercontent.com/faker-js/faker/main/src/locales/tr/person/first_name.ts
- Last names: https://raw.githubusercontent.com/faker-js/faker/main/src/locales/tr/person/last_name.ts
- Wikipedia: https://dumps.wikimedia.org/trwiki/latest/trwiki-latest-pages-articles.xml.bz2
- Word sequences: manually curated Turkish sequence lists in `src/wordSequences.json`
